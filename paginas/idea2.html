Problemática

- Excesiva Demanda del Servicio.

- Falta de Tiempo.

- Carencia de Automatización

- Bajo Control

- Poco Nivel Tecnológico

- Poca confianza de los Apoderados

- Desconocimiento de la ruta a seguir

Estas son las principales razones por la cual el trabajo para una flota de furgones escolares es complicado.

Sistema de Bus Escolar

El objetivo principal es poder ocupar las alternativas Tecnológicas dentro de una flota de furgones escolares.


Servirá para la Gestión de ésta y para la tranquilidad de los apoderados.



¿Cómo lograrlo?

- Se implementará una página WEB

- Se almacenarán los datos de los choferes, estudiantes y tutores

- Se va a tabular los datos contables de los pagos

- Listará los antecedentes de los choferes

- Se podrá localizar el bus mediante GPS
